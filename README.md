# Bootcamp Programacion de Cero a POO

## Descripcion
Este repositorio corresponde al bootcamp **Programacion de Cero a POO**, un curso virtual en vivo orientado a estudiantes que estan iniciando en programacion y deseando construir una base solida antes de avanzar hacia Programacion Orientada a Objetos.

El enfoque del bootcamp sigue una progresion academica practica:
**logica de programacion -- diagramas -- pseudocodigo -- C++ estructurado -- Java POO.

A lo largo del curso, se trabaja con clases en vivo, ejercicios practicos, tareas autonomas, evaluaciones y proyectos integradores, mantiendo siempre una metodologia centrada en comprender primero la logica, luego representarla y finalmente implementarla en codigo.

## Objetivo del respositorio
Este repositorio tiene como finalidad organizar y almacenar el material, ejercicios, practicas y proyectos desarrollados durante el bootcamp, ademas de servir como evidencia del proceso de aprendiza mediante el uso de Git y GitHub.

## Contenido del curso:
- logica y resolucion de problemas
- diagramas de flujo 
- pseudocodigo en PSeint 
- introduccion a C++
- variables, operadores y entrada/salida.
- estructuras condicionales
- estructuras repetitivas
- arreglos
- funciones y modularizacion 
- depuracion y correcion de errores
- introduccion a Git y GitHub
- prepracion de mini Proyecto en C++

## Estructura del repositorio 
La estructura puede variar segun el avance del curso, pero en general este repositorio puede incluir elementos como:

- clases -- material de apoyo o recursos vistos en clase
- ejercicios -- ejercicios desarrollados en las sesiones 
- tareas -- tareas enviadas para practica autonoma.
- proyectos -- mini-proyecto o trabajos integradores. 
- README.md -- presentacion general del respositorio.

## Metodologia de trabajo
Durante el curso se promueve un flujo de trabajo ordenado. Esto incluye:

- desarrollar ejercicios y proyectos de forma progresiva
- registrar avances mediante commits
- publicar cambios en GitHub
- mantener estructura clara en los repositorios 
- documentar de forma basica los proyectos mediante README.

El objetivo no es solo llegar a un resultado final, sino tambien evidenciar el proceso de contruccion del trabajo.

## Herramientas utilizadas.
Durante el curso se utilizan principalmente las siguientes herramientas:

- **Pseint** para pseudocodigo
- **Visual Studio Code** como editor principal
- **Git** para control de versiones 
- **GitHub** para repositorio y evidencias 
- **C++** como lenguaje principal en la etapa estructurada
- **Java** para la etapa posterior de Programacion Orientada a Objetos.
